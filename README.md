# baisahukam.29 — Rajputi Poshak & Jewellery

An elegant, fully-responsive e-commerce catalog web application custom-built for showcasing and managing exquisite traditional Rajputi Poshaks and royal jewellery. Combining a visually stunning, user-friendly frontend with a robust, secure administrative dashboard, this application provides an interactive experience for customers and seamless inventory control for store administrators.

## 🚀 Live Demo
Experience the live application here: [baisahukam.29 Live Website](https://radhapawarofficial.github.io/baisahukam.29-Rajputi-Poshaks-Jewellerys/)

---

## 🌟 Key Features

### 🛍️ Customer Experience
* **Thematic Design:** A warm, culturally aligned visual aesthetic styled with Tailwind CSS.
* **Interactive Catalog:** Customers can easily browse products categorized for intuitive navigation.
* **Detailed Lightbox Zoom:** High-resolution product images open in an interactive zoom modal for closer inspection.
* **Customer Media Gallery:** Real-life client photos and MP4 videos showcasing products.
* **Direct WhatsApp Integration:** A persistent floating action button connects customers directly with support to inquire about specific items.
* **Simple Checkout:** Instant order submission that routes customer details directly to the admin backend.

### 🔐 Administrative Dashboard
* **Firebase Authentication Guard:** Highly secure admin login panel restricted to authorized administrative emails.
* **Category Control:** Create, update, and delete product categories on the fly.
* **Inventory Management:** Add new items with multiple image uploads (hosted on Firebase Storage), configure pricing details (MRP vs. Selling Price), update descriptions, and toggle stock availability.
* **Media Gallery Controller:** Seamlessly upload client appreciation photos and promotional MP4 videos to the public gallery.
* **Live Order Tracker:** Monitor incoming customer inquiries and orders in real-time with automated timestamps, pricing calculations, and discount tracking.

---

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+), Tailwind CSS (CDN), Font Awesome Icons
* **Backend-as-a-Service (BaaS):** Firebase
  * **Firebase Auth:** Handles secure user authentication.
  * **Cloud Firestore:** Real-time, scalable NoSQL database managing products, categories, and client orders.
  * **Firebase Storage:** Cloud bucket hosting high-resolution product and gallery media uploads.
* **Hosting & Deployment:** GitHub Pages
